# C
#### Switch Statement 
[Source](https://www.geeksforgeeks.org/c-switch-statement/)
```c
switch(expression)
{
case value1: statement_1;
             break;
case value2: statement_2;
             break;
case value_n: statement_n;
              break;
default: default_statement;
}
```
